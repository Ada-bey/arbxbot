01

BingX'te API Key Oluştur

BingX → Hesabım → API Yönetimi → API Oluştur

02

Anahtarları Gir ve Kaydet

API Key + Secret Key'i yukarıya yapıştır → Kaydet → Test Et
"Bağlantı başarılı" mesajı görünce hazırsın.

03

Önce Simülasyonda Test Et

Simülasyon açık → ▶ Botu Başlat → Dashboard'dan fırsatları izle.
Gerçek emir gönderilmez, para harcanmaz.

04

Canlı Modu Aç

Simülasyon Kapat → ▶ Botu Başlat
Bot fırsat bulunca otomatik olarak 3 emir gönderir ve kâr eder.
